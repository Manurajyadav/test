# Specification

## Summary
**Goal:** Upgrade LevelUp app with enhanced dark theme, animated Welcome screen, AI chat assistant, and competitive leaderboard.

**Planned changes:**
- Apply comprehensive dark theme with #0a0a0f background, #1a1a2e cards, purple (#7c3aed) accents, glowing effects on cards and XP bar
- Upgrade Welcome screen with animated purple/blue gradient background, particle effects, glowing text, and dramatic subtitle
- Enhance quest cards with purple border glow, satisfying completion animations, +50 XP popup notifications, and fire emoji streak indicators
- Build AI Chat screen with SYSTEM assistant (glowing purple orb avatar), Gemini API integration, hardcoded opening messages, and typing indicators
- Build Leaderboard screen with 10 sample players, gold/silver/bronze glow effects for top 3 ranks, and purple highlight for current player "Shadow"

**User-visible outcome:** Users experience a visually stunning dark-themed app with animated effects, can chat with an AI assistant named SYSTEM for guidance, view their ranking against other players on a leaderboard, and receive satisfying feedback when completing quests.
